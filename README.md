# ORION PRO

AI-powered customer experience platform for South African businesses.

Built on Cloudflare Pages + Hono + WhatsApp API. Serving doctors, hotels, restaurants, and legal practices in KwaZulu-Natal.

## Features

- AI receptionist (Naledi) — 24/7 WhatsApp chat, appointment booking, FAQ
- Smart routing — knows when to answer and when to escalate to a human
- Multi-industry — medical, hospitality, legal, retail
- South African first — built for SA businesses by SA developers

## Tech Stack

- **Frontend:** Cloudflare Pages, HTML/CSS/JS
- **API:** Hono (Cloudflare Workers)
- **Database:** Cloudflare D1 (SQLite)
- **AI:** Z.AI (SiliconFlow) + Cloudflare Workers AI
- **Messaging:** WhatsApp Business API (Puppeteer → Meta Cloud API)
