Key Endpoints

POST/predictionsCreate prediction. Sync with Prefer: wait (up to 60s). Webhooks + SSE streaming supported.
GET/predictions/{id}Get status + output. Poll until succeeded/failed/canceled.
GET/predictionsList predictions (paginated, 100/page). Filter by created_after/before.
POST/predictions/{id}/cancelCancel a running prediction.
POST/models/{owner}/{name}/predictionsRun official model by name (no version ID needed).
GET/models/{owner}/{name}Get model details + OpenAPI schema + run count.
GET/modelsList public models. Sort by created_at.
POST/models/{owner}/{name}/versions/{id}/trainingsFine-tune a model. Saved to destination.
POST/deploymentsCreate scaling deployment (min/max instances, hardware SKU).
GET/collections/{slug}Get curated model collection.
GET/hardwareList hardware SKUs: CPU, T4, A40, L40S, H100.
GET/accountGet authenticated account info.

Rate Limits

Create prediction: 600 req/min. All other: 3,000 req/min. 429 on throttle.

Webhooks

webhook URL + webhook_events_filter (start, output, logs, completed). Output/logs throttled to 1 per 500ms.

Sync Mode

Prefer: wait=60 — keep open up to 60s. Cancel-After: 30s — auto-cancel long predictions.

Key Endpoints

POST/v2/scrapeScrape page (markdown/HTML/JSON/screenshot). 1 credit. JSON mode with schema.
POST/v2/searchWeb search + full page extraction. 2 credits/10 results. Domain/source/location filters.

GET/v2/crawl/{id}Poll crawl status + results.
POST/v2/mapDiscover URLs on a site. 1 credit/page. Optional search filter.
POST/v2/interactBrowser automation. 2 credits/min. AI prompt or code (bash/python/node).
POST/v2/parseParse PDF/DOCX/XLSX to markdown.
POST/v2/monitorSchedule recurring scrape + change detection. 1 credit/check.
POST/v2/agentAutonomous web research agent. Preview: 5 runs/day free.
